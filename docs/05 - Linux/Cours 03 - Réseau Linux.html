Réseau Linux

Voir aussi : Cours - DNS · Cours 02 - Administration Linux

Configuration IP

Afficher les interfaces :

ip addr

Afficher les routes :

ip route

Commandes utiles

Commande Description
ip addr Configuration IP
ping Tester la connectivité
ss -tulpn Connexions réseau
traceroute Chemin réseau
hostname Nom de la machine

DNS

Afficher les serveurs DNS :

cat /etc/resolv.conf

Tester la résolution :

nslookup google.com
dig google.com

SSH

Connexion distante :

ssh utilisateur@serveur

Copie de fichiers :

scp fichier utilisateur@serveur:/home/utilisateur

Le service SSH est généralement fourni par OpenSSH.

Pare-feu

Selon la distribution :

Ubuntu :

ufw status

Red Hat :

firewall-cmd --list-all
