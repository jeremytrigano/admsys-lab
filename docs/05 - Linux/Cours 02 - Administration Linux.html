Utilisateurs et groupes

Les utilisateurs sont enregistrés dans :

  • /etc/passwd
  • /etc/shadow

Les groupes sont enregistrés dans :

  • /etc/group

Commandes principales :

adduser utilisateur
userdel utilisateur
passwd utilisateur
groups utilisateur
id utilisateur

Permissions

Affichage :

ls -l

Exemple :

-rwxr-xr--

Correspondance :

  • r : lecture
  • w : écriture
  • x : exécution

Modifier les permissions :

chmod 755 fichier
chmod 644 fichier

Changer le propriétaire :

chown utilisateur fichier

sudo

Permet d'exécuter une commande avec les privilèges administrateur.

sudo commande

Gestion des paquets

Debian / Ubuntu :

apt update
apt upgrade
apt install paquet
apt remove paquet

Rocky / AlmaLinux :

dnf install paquet
dnf update
dnf remove paquet

Processus

Lister :

ps
ps aux
top

Arrêter :

kill PID
kill -9 PID

Cron

Planification de tâches automatiques.

Éditer les tâches :

crontab -e

Afficher :

crontab -l

Bash

Le shell Bash permet d'automatiser des tâches.

Exemple :

#!/bin/bash
echo "Bonjour"
